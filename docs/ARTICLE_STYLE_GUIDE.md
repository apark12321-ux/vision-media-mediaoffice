# 생활경제저널 기사 톤/문맥 구조 가이드

생활경제저널의 기사는 광고문, 블로그 후기, 업체 소개문이 아니라 독자가 상황을 이해하고 판단할 수 있도록 돕는 생활경제 기사체를 기준으로 작성한다.

## 1. 기본 톤

- 담백한 기사체를 사용한다.
- 과장된 홍보 문구를 피한다.
- 독자가 확인해야 할 기준을 설명한다.
- 업체 장점만 나열하지 않고 배경과 맥락을 함께 쓴다.
- 특정 결과를 보장하는 표현을 쓰지 않는다.
- 제휴 콘텐츠는 반드시 제휴 표시를 유지한다.

## 2. 일반 기사 구조

```text
[리드]
기사의 핵심 변화 또는 문제 제기를 2~3문장으로 제시한다.

[배경]
이 이슈가 나온 배경, 최근 흐름, 독자가 알아야 할 기본 정보를 설명한다.

[현장 맥락]
업계·지역·소비자 관점에서 실제로 어떤 변화가 있는지 풀어쓴다.

[독자 영향]
이 내용이 소비자, 보호자, 학부모, 사업자에게 어떤 의미인지 설명한다.

[전망과 과제]
앞으로 확인할 점, 남은 과제, 주의할 표현을 담백하게 정리한다.
```

## 3. 브랜드 인터뷰 구조

```text
[리드]
이 브랜드를 소개해야 하는 이유와 독자가 먼저 알아야 할 핵심을 2~3문장으로 쓴다.

[브랜드 배경]
업체가 어떤 지역·업종·고객층에서 활동하는지 설명한다.

[운영 철학]
대표자 또는 운영자의 기준, 고객을 대하는 방식, 서비스 원칙을 담는다.

[현장 맥락]
해당 업종에서 고객이 실제로 비교하는 기준과 불안을 설명한다.

[독자 체크포인트]
독자가 업체를 선택하기 전 확인하면 좋은 기준을 정리한다.

[정리]
과장 없이 브랜드의 의미와 향후 과제를 담백하게 마무리한다.
```

## 4. 보도자료 재가공 구조

```text
[리드]
발표의 핵심 사실을 육하원칙에 맞춰 2~3문장으로 정리한다.

[발표 내용]
기관·기업이 밝힌 주요 내용을 구체적으로 설명한다.

[사업 배경]
왜 이 발표가 나왔는지, 관련 시장·지역·독자 맥락을 설명한다.

[향후 계획]
후속 일정, 적용 범위, 기대 효과를 사실 중심으로 쓴다.

[정리]
독자가 확인해야 할 의미와 유의점을 덧붙인다.
```

## 5. 작성 전 체크

- 요약문이 기사 핵심, 배경, 독자 영향을 포함하는가?
- 본문이 최소 5개 문맥 단락으로 구성되어 있는가?
- 리드, 배경, 현장 맥락, 독자 영향, 정리가 보이는가?
- 다만, 특히, 이에, 한편, 따라서 등 문맥 연결어가 자연스럽게 들어갔는가?
- 광고주 제공 자료와 편집부 판단이 구분되는가?
- 제휴 콘텐츠인 경우 고지문이 표시되는가?
- 이미지 출처와 라이선스가 확인되었는가?

## 6. 피해야 할 문체

- “최고”, “유일”, “완벽”, “100%”, “보장” 등 단정형 홍보 문구
- 업체 장점만 나열하는 소개문
- 배경 설명 없이 결론만 말하는 짧은 문단
- 독자에게 어떤 의미인지 설명하지 않는 정보 나열
- 출처가 불분명한 이미지 또는 자료 사용
